## 书评

此书于我而言具有重大意义，某种意义上来说，它是我在CS道路上的启蒙书籍。

当年在我备战高考及考虑大学专业方向选择的时候，忘了是在知乎还是贴吧上看到关于计算机编程的语言进阶文章，作者建议是先学C，再到C++及其他。

此书购于2015年，是我进入大学前的那个暑假。它有一个缩写“K&R”，又名标准C。

大一学校用了1学年的时间去学习C。

大二上学期开始选方向 Java 和 C#。

选了 Java 也去旁听了 C#。
